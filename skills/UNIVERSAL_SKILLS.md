# Universal Skills

这是统一的 AI Coding Skills 索引，从 ~/universal-skills/categories 加载。

## 可用分类

- **coding** - 编程规范、TDD、设计模式
- **debugging** - 调试方法论
- **refactoring** - 重构技能  
- **architecture** - 架构设计
- **workflow** - 工作流管理

## 使用方法

在 Gemini 中询问：
- "查看 TDD workflow skill"
- "使用 systematic debugging"
- "应用 minimalist refactorer"

完整内容位于 ~/universal-skills/categories/
